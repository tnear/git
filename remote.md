# remote

`git-remote` - Manage set of tracked repositories

## Show name
```
$ git remote
origin
```

## Verbose
Use `-v` to see verbose output.
```
$ git remote -v
origin  https://github.com/tnear/git.git (fetch)
origin  https://github.com/tnear/git.git (push)
```
